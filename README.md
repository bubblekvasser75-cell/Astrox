# LEAN HOUSE — краш-игра (Telegram Mini App)

Краш-игра «ракетка», где вместо ракеты летит стакан. Монеты LH, список игроков,
авто-раунды с отсчётом 10 секунд, кнопка «Получить 1000 LH».

## Файлы
- `index.html` — сама игра (одним файлом, картинка стакана вшита).
- `bot.py` — Telegram-бот (aiogram), открывает игру.
- `fonts/` — сюда положи файлы шрифта Benzin (необязательно).

## Шрифт (Benzin Bold)
Игра использует шрифт `Benzin`. Если файлов нет — автоматически подхватывается
бесплатный похожий Unbounded (с Google Fonts). Чтобы поставить настоящий Benzin:
1. Купи/скачай `Benzin-Bold` и (желательно) `Benzin-ExtraBold`.
2. Конвертируй в `.woff2` (напр. на transfonter.org).
3. Положи файлы в папку `fonts/`:
   - `fonts/Benzin-Bold.woff2`, `fonts/Benzin-Bold.ttf`
   - `fonts/Benzin-ExtraBold.woff2`, `fonts/Benzin-ExtraBold.ttf`
Больше ничего менять не нужно — `@font-face` уже прописан в `index.html`.

## Как сделать мини-апп в Telegram

### Шаг 1. Выложи index.html на HTTPS
Web App работает только по https. Быстрые варианты:
- **GitHub Pages**: создай репо, залей `index.html`, включи Pages → получишь ссылку вида `https://логин.github.io/репо/`.
- **Netlify / Vercel / Cloudflare Pages**: просто перетащи папку — получишь HTTPS-ссылку.
- **Свой сервер**: nginx + сертификат (Let's Encrypt).

### Шаг 2. Создай бота у @BotFather
1. Открой @BotFather → `/newbot` → получи `BOT_TOKEN`.
2. (Можно без кода) `/newapp` → выбери бота → укажи название, картинку и **URL игры** (ссылка из шага 1).
   После этого игра уже открывается по прямой ссылке `t.me/имя_бота/имя_аппа`.
3. (Опционально) `/setmenubutton` → привяжи кнопку-меню к URL игры.

### Шаг 3. Запусти бота (чтобы была кнопка /start)
```bash
pip install aiogram==3.13.1
export BOT_TOKEN="123456:ABC..."
export WEBAPP_URL="https://твой-хост/index.html"
python bot.py
```
Бот даёт команду `/start` с кнопкой «Играть», кнопку внизу экрана и синюю
Menu-кнопку Web App.
